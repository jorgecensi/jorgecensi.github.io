Binary Puzzle

Fill the grid with 0s and 1s. Pick an input mode, or stay on Cycle to rotate values on click.

Input mode

Game Rules

  • Each row and column should contain an equal number of 0s and 1s.
  • No more than two of the same number can be adjacent.
  • Each row and each column should be unique.
